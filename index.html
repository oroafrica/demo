Create React application with the following functionality.

  1. Project name: Sketcher
  2. Render SVG file on canvas.
  3. Iterate various SVG elements.
  4. Add customizable text using supplied font.
  5. Add custom image via file input.
  6. Export canvas to png/jpg
  7. Canvas element controls optional.
  8. Apply conventionaljavascript design methodology

Supplied assets

  • Library: Fabric
  • Fonts: z_boli
  • SVG: shapes

Required components

  • Canvas 450x450
  • Button: [share] Canvas to image
  • Button: [left] load previous SVG shape
  • Button: [right] load next SVG shape
  • Button: [text] add customised text to canvas
